# Live Data Dashboard with Streamlit (UCL 2015–16)

This dashboard explores UEFA Champions League 2015–16 matches.   
I found public github football dataset and processed it with pandas to build interactive charts in Streamlit

## Dataset

- Source (github): https://github.com/footballcsv/europe-champions-league  
- Direct file: https://github.com/footballcsv/europe-champions-league/blob/master/2015-16/champs.csv
- Dataset file in: `Live-Data-Dashboard-with-Streamlit
/data/dataset-unique(sport-football-2015-16).csv`

## Why I chose this dataset

I picked Champions League data because it is unique (sports) to analyze with filters.  
It has clear events (matches), measurable stats (goals), and multiple stages/rounds for comparisons.
Every requirements are touched and checked by me.
